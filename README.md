# 📊 Sales & Inventory Analytics using IBM Cognos

## 📌 Project Overview
This project focuses on analyzing supermarket sales and inventory data using IBM Cognos Analytics. The goal is to generate meaningful insights that help in business decision-making, improve sales performance, and reduce stockouts.

---

## 🚀 Key Features
- 📈 Sales Summary Report (Category & Sub-category wise)
- 💰 Profit & Profit Margin Analysis
- 🛑 Slow-Moving Products Identification
- 📊 Weekly Sales Trends (Bar Chart & Line Chart)
- 🧾 KPI-based Store Manager Dashboard
- 📉 Inventory & Stockout Impact Analysis

---

## 🛠 Tools & Technologies
- IBM Cognos Analytics  
- Data Visualization  
- Business Intelligence Reporting  

---

## 📊 Reports & Dashboards
### 1. Sales Summary Report
- Displays Total Sales, Profit, and Profit Margin %
- Grouped by Category and Sub-category  

### 2. Slow-Moving Products Report
- Identifies products with low sales (below 50 units)
- Uses conditional formatting for better visualization  

### 3. Weekly Sales Trends
- Interactive visualization using Bar and Line charts  

### 4. Store Manager Dashboard
- KPI Tiles (Sales, Profit, Stockouts)
- Category-wise Sales Breakdown (Pie Chart)
- Outlet-wise performance (Table)

### 5. Inventory & Stockout Analysis
- Shows relationship between stock availability and sales performance
